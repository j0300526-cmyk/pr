## 🐛 문제 해결

### 401 Unauthorized 오류
- 테스트 사용자가 생성되지 않았을 수 있습니다
- `python init_data.py` 실행 후 다시 시도하세요

### 로그인이 안 되는 경우
- 이메일: `test@example.com`
- 비밀번호: `test1234` (정확히 입력)
- 서버가 실행 중인지 확인
